smooth

Smooth with a moving average.

Syntax

[o s] = smooth(x,m,wn)

Description

[o s] = smooth(x,m,wn) performs smoothing on x by averaging in a sliding window of wn size. m describes the data type, i.e. 'linear' or 'circular'. Output is returned in o with standard error of the smoothing in s.
