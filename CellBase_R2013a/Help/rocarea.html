rocarea

Receiver-operator characteristic.

Syntax

d = rocarea(x,y)
[d p se] = rocarea(x,y,'bootstrap',n)

Description

d = rocarea(x,y) calculates area under ROC curve (d) for the samples x and y.

[d p se] = rocarea(x,y,'bootstrap',n) calculates permutation test p-value and bootstrap standard error (n, number of bootstrap samples).

Optional input parameter-value paits (with default values):

'bootstrap', 0 - size of bootstrap sample; 0 corresponds to no bootstrap analysis

'transform', 'none' - 'swap': rescales results between 0.5 - 1; 'scale' - rescales results between -1 to 1

'display', false - controls plotting of the ROC curve
