listtag

List CellBase tags.

Syntax

l = listtag(str)

Description

l = listtag(str) returns a cell array containing the specified tag from CellBase. Options for str: 'properties', 'analysis', 'cell', 'rat'/'animal', 'session', 'tetrode'. 'properties' and 'analysis' are searched in ANALYSES, others are searched in CELLIDLIST. The 'allsession' option includes sessions without cells (e.g. behavior only) by searching the directory tree.
